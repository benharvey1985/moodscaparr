**Migration Pattern (applies to all 5 pages):**
Each client page follows the same 4-step refactoring after being moved:

1. **Delete** `import { Header } from "@/components/header"` and every `<Header user={...} />` usage
2. **Delete** the entire `useEffect` block that calls `authClient.getSession()`, the associated `session` + `sessionLoading` state variables, and the `useRouter` import (unless router is still used elsewhere)
3. **Delete/Simplify** the `if (sessionLoading) { return ... }` loading branch ŌĆö collapse into main content or remove entirely (pages that load data via TanStack Query already have their own loading states)
4. **Replace** `<div className="flex min-h-screen flex-col">` ŌåÆ `<div>` and `<main ...>` ŌåÆ `<div>` ŌĆö remove `pb-12` from the content div. The `<main>` tag and `pb-16` are provided by AppLayoutClient.
