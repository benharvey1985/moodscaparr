## Goal

**As a** user, **I want to** customize my profile, manage my data, and as an admin, manage the instance — **so that** the app adapts to my preferences and stays healthy.

<objective>
Build user profile settings (edit, data export/restore), achievement summary view, and a full admin panel (dashboard, user management, SSO config, invite codes).
</objective>

Create app/api/admin/stats/route.ts:
- GET /api/admin/stats: admin-only, return { totalUsers, activeUsers (entries in last 30d), totalEntries, avgStreak, registrationTrend (last 12 months), moodDistribution (POSITIVE/NEUTRAL/NEGATIVE percentages) }
- Require admin role

Create app/api/admin/users/route.ts:
- GET /api/admin/users: admin-only, return paginated user list with search and role filter. Response: { users: Array<{ id, name, email, role, banned, entryCount, lastEntryDate }>, total, page, limit }
- Search across name, email

Create app/api/admin/users/[id]/route.ts:
- PUT /api/admin/users/:id — update role (promote/demote), ban/suspend status, delete
- DELETE with soft-delete (set banned=true, banReason="deleted", banExpires=30d) for undo
- DELETE already-banned users: actually delete
- Auth check (admin only)

Create app/api/admin/sso/route.ts:
- GET /api/admin/sso: list SSO providers (Google, GitHub) with enabled status
- PUT /api/admin/sso/:provider: toggle enabled
- Auth check (admin only)

Create app/api/admin/invite-codes/route.ts:
- GET /api/admin/invite-codes: list all codes
- POST /api/admin/invite-codes: generate new code with maxUses, expiresAt
- DELETE /api/admin/invite-codes/:id: revoke code (set active=false)
- Auth check (admin only)
