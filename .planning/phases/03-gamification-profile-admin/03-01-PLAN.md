## Goal

**As a** user building a mood tracking habit, **I want to** earn badges as I hit milestones, see my progress toward each badge, and celebrate unlocks — **so that** I stay motivated and engaged.

<objective>
Build a complete achievement system: 14 badges (milestone/streak/exploration/special), progress tracking, confetti on unlock, toast notifications, and overall completion display.
</objective>

Add Achievement model to prisma/schema.prisma:
```
model Achievement {
  id        String   @id @default(cuid())
  userId    String
  badgeId   String
  unlockedAt DateTime @default(now())
  progress  Int      @default(0)
  maxProgress Int    @default(1)

  user      user     @relation(fields: [userId], references: [id], onDelete: Cascade)
  
  @@unique([userId, badgeId])
  @@index([userId])
}
```

Create types/achievements.ts:
- Export const ACHIEVEMENTS array with 14 badge definitions:
  - Each: { id: string, category: "milestone"|"streak"|"exploration"|"special", icon: string, title: string, description: string, condition: string, maxProgress: number }
  - Milestone: "first-entry" (1), "double-digits" (10), "half-century" (50), "century" (100)
  - Streak: "week-warrior" (7), "month-master" (30), "bimonthly" (60), "endurance" (30 longest)
  - Exploration: "mood-explorer" (19 distinct moods), "weather-watcher" (7 weathers), "activity-diver" (5+ in one), "reflection-king" (50 total)
  - Special: "early-bird" (before 9am), "night-owl" (after 10pm)
- Export function checkAchievements(userId, context): queries user's entries + existing achievements, returns newly unlocked badges

Create lib/achievements.ts:
- Server-side functions:
- async function getUserAchievements(userId): returns all achievements for user with badge metadata
- async function checkAndUnlockAchievements(userId): checks all 14 conditions against DB, unlocks any new ones, returns newly unlocked array
- async function hasEntryTodayBeforeHour(userId, hour): helper for special badges
- async function distinctMoods(userId): count of distinct moodIndex values across entries
- async function totalReflections(userId): count of non-null reflection fields

Create app/api/achievements/route.ts:
- GET /api/achievements: return user's achievements + overall completion percentage + all 14 definitions with progress
- Auth check

Create app/api/achievements/check/route.ts:
- POST /api/achievements/check: trigger check for newly unlockable badges
- Auth check
- Used after mood log mutations to check in real-time
