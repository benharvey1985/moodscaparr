## Goal

**As a** new user, **I want to** get a guided tour on first login and have a quick way to give feedback — **so that** I understand the app's capabilities and can report issues or suggest improvements easily.

<objective>
Build a 3-step onboarding tour (shown on first login, skippable) and a floating feedback button with bug report/feature suggestion dialogs that create GitHub issues.
</objective>
